## EC2 Windows

- リモートデスクトップ接続する
- Session Managerで接続する
- ec2
  - windows
- 毎日0時に自動停止する

### Test

- 以下の手順でリモートデスクトップ接続する
  - インスタンスに接続
  - RDP クライアント
  - Fleet Manager を使用して接続する
  - パスワードかキーペアで認証する
- 自動停止について、GUIからcron式を編集して5分ごとにイベント発火とかにしたら確認しやすい

### Tips

- ec2 のインスタンスはMEM 4GBとかそこそこいいのを使うと良さそう
  - 無料利用枠のインスタンスではリモートデスクトップは性能的に厳しい。めっちゃ重い
